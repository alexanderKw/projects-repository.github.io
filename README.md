                                              POKEDEX-ИНСТРУКЦИЯ
                                            ======================

http://projects-repository.github.io/   --   ссылка для запуска приложения при помощи сервиса GitHub Pages.

При запуске приложения, на страницу загружается список, состоящий из 12-и карточек Покемонов и кнопка "Load More".

При клике на кнопку "Load More", загружается и отображается другая часть списка Покемонов.

При клике на отдельную карточку Покемона, в правой части экрана отображается ее увеличенная копия с подробными характеристиками персонажа.

При клике на следующую(другую) карточку, данные нового персонажа заменяют данные предыдущего.

Карточка с подробными характеристиками имеет фиксированное положение на экране.

Приложение адаптировано под размеры экранов настольных компьютеров, и под размеры экранов мобильных устройств.
